В файле задается автомат в следующем виде:
Q = {<состояния>}, A = {<символы алфавита>}, q = <начальное состояние>, F = {<конечные состояния>}, 
D = {<правила переходов из состояния в состояние в виде d(<текущее состояние>, <символ>) = <результирующее состояние>>}

Например:
Q = {1, 2}, A = {a, b}, q = 1, F = {2}, D = {d(1; a) = 2, d(1; b) = 1, d(2; a) = 2, d(2; b) = 2}

Результат сохраняется в файл "result.txt"

Скрипт запускается на Ubuntu и Windows с помощью команды python.
